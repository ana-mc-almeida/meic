What is **reflection**?

What is **introspection**?
- The ability to *examine* itself

What is **intercession**?
- The ability to *modify* itself
- Java does not allow intercession

What is **reification**?
- Pre-condition for reflection
- Stuff need to become values: methods, classes, parameters, etc
- Entities that represente concepts

What is **structural reflection**?
- Reflection about the structure of the program

What is **behavioral reflection**?
- Reflection about the behavior/executation of the program

> behavioral reflection is harder than structural reflection
